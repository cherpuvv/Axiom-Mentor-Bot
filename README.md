# Axiom — Mentor Bot

Telegram-бот для сбора структурированной двусторонней обратной связи
между стажёрами и наставниками в рамках пилотного проекта.

## Статус
Beta заморожена

## Функционал (beta)
- регистрация пользователей (стажёр / наставник)
- чек-лист стажёра (оценка процесса стажировки)
- чек-лист наставника (оценка процесса обучения стажёра)
- сохранение результатов 
- экспорт данных (CSV)

## Технологии
- Python
- Telegram Bot API
- SQLite
